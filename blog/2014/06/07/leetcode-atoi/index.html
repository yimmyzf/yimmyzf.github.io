Leetcode String to Integer

注意的几点：

* 字符串为空
* 全是空格
* 字符串前的空格要忽略掉
* 对‘+’和‘-’的处理
* 遇到非数字的字符就返回当前值而不是返回0
* int的范围  -2147483648~2147483647

【判断与int最值比较时记得*signal】

又用Python写了一遍，遇到的小问题：

* 空字符串str==''
* elif
* 判断条件中要加入i<len(str)
* 需要用ord()将字符转换成ASCII码
